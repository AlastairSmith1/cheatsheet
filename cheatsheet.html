Cheat Sheet

This is a cheat sheet to help learning HTML and CSS

HTML Tag helper

Tags Usage Explaination
!DOCTYPE html Beginning of the document Tells the browser that the document type is HTML
html Beginning of the doc after the DOCTYPE declaration This element is the root element of an HTML page
head First section of the doc Element contains meta information about the HTML page
title First part of the head section Element specifies a title for the HTML page

CSS syntax helper

Syntax Where it's used What is does
; At the end of a line Used to indicate the end of a line
selector At the beginning of a new rule Indicates the element which you want to style. A . is used as a prefix for a class and a # for an ID
property After the squiggly brackets Indicates how you are going to style the element.
value After a colon opposite the property Indicates the how the property is going to be changed.
